# 金融应用Python编程教学范例

一个基于网页操作的Python编程金融应用教学范例项目，包含10个金融应用主题和60个详细案例。

## 项目结构

```
financial_education_app/
├── venv/                    # Python虚拟环境
├── app.py                   # Flask应用主程序
├── requirements.txt         # 项目依赖
├── templates/               # HTML模板文件
│   ├── base.html          # 基础模板
│   ├── index.html         # 首页
│   ├── topic_detail.html  # 主题详情页
│   └── category_detail.html # 分类详情页
├── static/                  # 静态资源
│   ├── css/               # 样式文件
│   ├── js/                # JavaScript文件
│   └── images/            # 图片文件
└── examples/               # 代码示例目录
```

## 功能特点

### 三级网页结构

1. **首页 (index.html) - 金融应用主题**
   - 展示10个金融应用的Python编程示范案例主题
   - 每个主题包含图标、名称和简要描述

2. **主题详情页 (topic_detail.html) - 分类主题**
   - 每个主题下的第二级网页规划6个分类主题
   - 数据获取、数据处理、数据分析、可视化、机器学习、实战案例

3. **分类详情页 (category_detail.html) - 具体代码**
   - 展示具体代码和代码说明
   - 代码高亮显示
   - 提供复制和运行代码功能

### 10个金融应用主题

1. **股票分析工具** - 股票数据获取、分析和可视化
2. **量化交易策略** - 算法交易和策略回测
3. **风险管理系统** - 风险评估和投资组合优化
4. **财务报表分析** - 财务数据处理和报表解读
5. **加密货币分析** - 区块链数据和加密货币追踪
6. **债券计算工具** - 债券定价和收益率计算
7. **房地产投资分析** - 房产估值和投资回报计算
8. **金融风险管理** - VaR计算和风险度量
9. **外汇交易系统** - 汇率分析和交易信号
10. **数据分析可视化** - 金融数据可视化和图表

## 环境搭建

### 1. 创建虚拟环境

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 运行应用

```bash
python app.py
```

### 4. 访问应用

在浏览器中访问 `http://localhost:5000`

## 使用说明

### 代码运行

- 每个代码示例都提供完整的可运行代码
- 点击"复制代码"按钮可以复制完整代码到剪贴板
- 点击"运行代码"按钮可以在浏览器控制台运行代码
- 代码使用Prism.js进行语法高亮

### 技术栈

- **后端**: Flask (Python)
- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **数据获取**: yfinance
- **数据处理**: pandas, numpy
- **数据分析**: scikit-learn, scipy
- **可视化**: matplotlib, seaborn
- **代码高亮**: Prism.js
